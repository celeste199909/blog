    预编译四部曲

    预编译发生在函数时，预编译发生在函数前一刻；

    1. 创建AO对象Activation Object，即执行期上下文
    2. 找形参和变量声明，将变量和形参名作为AO的属性名，值为undefined
    3. 将实参值和形参相统一
    4. 在函数体里面找函数声明，值赋予函数体

    预编译不仅发生在函数内，还发生在全局；

    因为全局没有参数，所以全局的预编译是：

    1. 创建GO对象（window），即执行期上下文
    2. 找形参和变量声明，将变量和形参名作为AO的属性名，值为undefined
    3. 找函数声明，值赋予函数体
